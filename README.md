| Prefix (type) | 설명                |
|---------------| ----------------- |
| `Feature:`    | 새로운 기능 개발         |
| `Fix:`        | 버그 수정             |
| `Refactor:`   | 리팩토링 (기능 변경 없음)   |
| `Style:`      | 스타일 (UI, CSS 등)   |
| `Docs:`       | 문서 변경             |
| `Test:`       | 테스트 코드 추가         |
| `Chore:`      | 설정, 빌드 수정 등 기타 작업 |
